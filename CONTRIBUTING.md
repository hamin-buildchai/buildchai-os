# Contributing

Early-stage. Thanks for the interest.

## What we want now
- **Issues**: bugs, questions, suggestions
- **Examples**: fork + share your own multi-agent Discord setup

## What we don't want yet
- Large PRs to core (still rapidly iterating — likely to rewrite)
- Dependency additions without discussion

## Running locally
Requires:
- Discord bot tokens for each agent role
- OpenClaw CLI (`npm i -g openclaw`)
- Python 3.11+

See `docs/architecture.md` for the layered setup.

## Style
- Shell scripts: bash 4+, `set -e`
- Python: 3.11, no heavy deps (we run on a NUC)
- Markdown: keep sections short, code examples over walls of prose
